# Residual: `PresheafOfModules.isLocallyInjective_whiskerLeft_of_W`

**Statement (TRUE).** Let `R : C·µí·µñ ‚•§ CommRingCat`, `J` a Grothendieck topology with
`J.WEqualsLocallyBijective Ab`. For an arbitrary `F : PresheafOfModules (R ‚ãô forget‚ÇÇ _ _)`
and `g : M ‚ü∂ N` with `J.W ((toPresheaf _).map g)` (locally bijective), the left-whisker
`F ‚óÅ g` is locally injective: `IsLocallyInjective J (F ‚óÅ g)`.

This is the SINGLE residual on which `tensorObj_assoc_iso` now depends (everything else
in the associator is closed, axiom-clean). Used (via `W_whiskerLeft_of_W` /
`W_whiskerRight_of_W`) only with `g = toSheafify` and `F = M.val`/`P.val`.

## Proof (ROUTE (d), Mathlib-blessed flatness-free technique)

The whole point: a `J.W`-morphism is a **stalkwise isomorphism**, and tensoring an
isomorphism by `id_F` is an isomorphism ‚Äî so `F ‚óÅ g` is a stalkwise iso, hence locally
bijective, hence locally injective. No flatness, no local triviality. (Contrast the
flat route `isLocallyInjective_whiskerLeft_of_flat`, which preserves injectivity ALONE
and so needs `Module.Flat.lTensor_exact` ‚Äî false for invertibles over non-affine opens.)

Concretely on the topological site of a scheme `X` (`C = Opens X`,
`J = Opens.grothendieckTopology X`):

1. **(d.1) Module-`J.W` ‚ü∫ stalkwise iso.** For `f` a morphism of presheaves of
   `R`-modules on `Opens X`, `J.W ((toPresheaf _).map f) ‚Üî ‚àÄ x, IsIso ((stalkFunctor Ab x).map ((toPresheaf _).map f))`.
   - `J.W = locally bijective = locally injective ‚àß locally surjective`
     (`WEqualsLocallyBijective`).
   - surjective half: bridge `CategoryTheory.Presheaf.IsLocallySurjective J` ‚ü∫
     `TopCat.Presheaf.IsLocallySurjective` then
     `TopCat.Presheaf.locally_surjective_iff_surjective_on_stalks`.
   - injective half: the analogue for `IsLocallyInjective` (Mathlib has only the
     SEPARATED/sheaf version `app_injective_iff_stalkFunctor_map_injective`; needs the
     locally-injective ‚ü∫ stalk-injective bridge for the `Opens X` topology).
   - Abstractly this is `ObjectProperty.IsConservativeFamilyOfPoints.W_iff` +
     `TopCat.hasEnoughPoints` (`Mathlib.Topology.Sheaves.Points`, 2026); the concrete
     stalk packaging is `TopCat.Presheaf.isIso_iff_stalkFunctor_map_iso`.

2. **(d.2) Stalk commutes with the relative module tensor.**
   `((A ‚äó·µñ B).presheaf).stalk x ‚âÖ A.stalk x ‚äó_{R.stalk x} B.stalk x`, naturally, so that
   `((F ‚óÅ g).presheaf).stalk x = id_{F_x} ‚äó ((g).presheaf).stalk x` under the iso.
   - `(A ‚äó·µñ B).obj U = A.obj U ‚äó_{R(U)} B.obj U` sectionwise
     (`PresheafOfModules.Monoidal.tensorObj_obj`).
   - stalk = filtered colimit over `{U ‚àã x}`; `tensorLeft`/`tensorRight` preserve
     filtered colimits over a module category; the ring varies, so this is
     "colimit of tensors ‚âÖ tensor of colimits over the colimit ring". Genuinely
     missing at the `PresheafOfModules` level.

3. **Conclude.** `g_x := (toPresheaf g).stalk x` is iso by (d.1). By (d.2),
   `(F ‚óÅ g)_x = id_{F_x} ‚äó g_x`, an iso (any functor preserves isos). So `F ‚óÅ g` is a
   stalkwise iso, hence (d.1 again) `J.W ((toPresheaf _).map (F ‚óÅ g))`, hence
   `IsLocallyInjective J (F ‚óÅ g)` (`GrothendieckTopology.W.isLocallyInjective`).
