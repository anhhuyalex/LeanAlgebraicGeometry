## Decisions identified

### Decision: Route A — assemble a global `Presentation M` from `QuasicoherentData`

- **Mathlib idiom**: NONE in the local→global direction. `SheafOfModules.QuasicoherentData`
  (Quasicoherent.lean:201) holds a cover `X : I → C` + per-member `(M.over (X i)).Presentation`.
  Its API only goes the *other* way or shuffles local data: `Presentation.quasicoherentData`
  (Pres → QCData), `QuasicoherentData.bind`/`shrink`/`localGeneratorsData`,
  `IsQuasicoherent.of_coversTop`. The global `GeneratingSections` structure
  (Generators.lean:45) has **no constructor from `LocalGeneratorsData`**. There is no
  "globally generated on affine/qcqs", no "sections generate a QC sheaf on affine".
- **Gap**: divergent-and-wrong-to-attempt. Route A is strictly HARDER than the stalk/section
  route: a global `Presentation` additionally needs *global generation by sections*
  (QC on affine ⟹ generated by global sections), which itself needs the same
  section-localization/affine-vanishing input plus a kernel-is-also-QC step.
- **Verdict**: NEEDS_MATHLIB_GAP_FILL (do **not** target Route A).

### Decision: Route B — `essImage`/`fromTildeΓ` directly via stalk-iso (recommended)

- **Mathlib idiom**: a sheaf morphism is iso iff iso on all stalks, plus fully-faithful
  reflects iso. Concretely:
  - `CategoryTheory.Functor.FullyFaithful.reflectsIsomorphisms` (ReflectsIso/Basic.lean:53)
    via the project's `SpecModulesToSheafFullyFaithful`:
    `IsIso M.fromTildeΓ ⟸ IsIso (modulesSpecToSheaf.map M.fromTildeΓ)`.
  - `TopCat.Sheaf.isIso_of_stalkFunctor_map_iso` (Topology/Sheaves/Stalks.lean:636) /
    `isIso_iff_stalkFunctor_map_iso` (:652): a `TopCat.Sheaf (ModuleCat R) (Spec R)` morphism
    is iso ⟸ iso on every stalk.
  - Source stalk is free: `tilde.toStalk` carries
    `IsLocalizedModule x.asIdeal.primeCompl (toStalk M x).hom` (Tilde.lean:131) ⟹
    `(tilde N).stalk p ≅ N_p` with `N = Γ(M,⊤)`.
  - `IsLocalizedModule.linearEquiv` (LocalizedModule/Basic.lean:923): two localizations of `N`
    at the same submonoid, related by a compatible map, are `≃ₗ` — turns "both sides localize
    `N`" into the component/stalk iso.
- **Gap**: divergent-with-cost — Mathlib supplies every *glue* lemma, but the **stalk-localization
  core** (`M.stalk p` is the localization of `Γ(M,⊤)` at `p.primeCompl`, derived from
  `QuasicoherentData`) is absent and is irreducible content.
- **Verdict**: NEEDS_MATHLIB_GAP_FILL (this is the cleanest assembly; the core sub-lemma is the gap).

### Decision: Route C — basis/section route (matches the project's existing building blocks)

- **Mathlib idiom**: same stalk machinery, but the workhorse is the **section-localization**
  lemma the project is already circling:
  `QC M ⟹ IsLocalizedModule (powers f) (restrict Γ(M,⊤) → Γ(M,D(f)))`.
  Given it, on each basic open D(f) the component of `modulesSpecToSheaf.map M.fromTildeΓ`
  is `N_f → Γ(M,D(f))`; source localizes via `isLocalizedModule_tilde_restrict` + `tilde.isoTop`,
  target via the section-localization lemma, and `Scheme.Modules.toOpen_fromTildeΓ_app`
  (Tilde.lean:230) makes the triangle commute, so `IsLocalizedModule.linearEquiv` gives the
  component iso. Basic opens are a basis ⟹ stalk-iso ⟹ `isIso_of_stalkFunctor_map_iso`.
- **Gap**: divergent-with-cost — identical irreducible core, packaged on sections instead of stalks.
- **Verdict**: NEEDS_MATHLIB_GAP_FILL. **Preferred for THIS project**, because the
  section-localization lemma is *also* the missing keystone
  `lem:qcoh_section_localization_basicOpen` and is reused by gap2; proving it once discharges both.

## The irreducible content (the actual gap)

All three routes funnel through ONE Mathlib-absent fact (Stacks tag 01HA / Hartshorne II.5.2):

> **For a quasi-coherent `M` on `Spec R` and `f : R`, the restriction `Γ(M,⊤) → Γ(M,D(f))`
> is `IsLocalizedModule (Submonoid.powers f)`** (equivalently `M.stalk p` is the localization
> of `Γ(M,⊤)` at `p.primeCompl`).

Proof skeleton from `QuasicoherentData` (genuine multi-step descent):
1. **Refine the QC cover to basic opens.** `PrimeSpectrum.isBasis_basic_opens`; restrict each
   `(M.over (X i)).Presentation` to a basic open `D(g_a) ⊆ X_i` via `Presentation.map` along the
   `pushforward`/`over` restriction functor (Quasicoherent.lean:179). `CompactSpace (PrimeSpectrum R)`
   (Spectrum/Prime/Topology.lean:291) ⟹ a **finite** subcover; the `g_a` generate the unit ideal.
2. **Locally `M` is a tilde sheaf.** On `D(g_a) = Spec R_{g_a}`, the local presentation +
   `isIso_fromTildeΓ_of_presentation` ⟹ `M.over (D g_a) ≅ tilde(N_a)`, `N_a = Γ(M, D(g_a))`,
   over the ring `Γ(Spec R, D(g_a)) = R_{g_a}` (`AlgebraicGeometry.isLocalization_basicOpen`,
   AffineScheme.lean:650).
3. **Sheaf equalizer + flat descent.** Sheaf condition over the finite cover {D(g_a)} gives
   `0 → Γ(M,⊤) → ∏ N_a → ∏ Γ(M, D(g_a g_b))`; localizing at `powers f` is exact
   (`IsLocalization.flat`, Flat/Localization.lean:36; localization commutes with finite limits)
   and matches the same equalizer for `Γ(M,D(f))`, yielding `Γ(M,⊤)_f ≅ Γ(M,D(f))`.
4. Stalk version: `M.stalk p = colim_{D(g)∋p} Γ(M,D(g)) = colim N_g = N_p`.

## Recommendation

gap1 is **genuine multi-step descent with no current Mathlib idiom**. Do NOT attempt a single
lemma or Route A (global `Presentation`). Target **Route C** and split the work as:

- **G1-core** (the gap): `QC M ⟹ IsLocalizedModule (powers f) (Γ(M,⊤) → Γ(M,D(f)))`. This is the
  keystone; build it from `QuasicoherentData` via the cover-refine → local-tilde → flat-equalizer
  skeleton above. Reuses `isLocalization_basicOpen`, `IsLocalization.flat`, `CompactSpace`.
- **G1-assemble** (cheap, all Mathlib glue): from G1-core, prove each basic-open component of
  `modulesSpecToSheaf.map M.fromTildeΓ` iso via `IsLocalizedModule.linearEquiv` +
  `isLocalizedModule_tilde_restrict` + `toOpen_fromTildeΓ_app`, then
  `isIso_of_stalkFunctor_map_iso` + `FullyFaithful.reflectsIsomorphisms` to land `IsIso M.fromTildeΓ`.

**Blueprint circularity to fix:** `lem:qcoh_section_localization_basicOpen` currently `\uses`
`lem:qcoh_affine_isIso_fromTildeΓ` (gap1) AND the project's
`isLocalizedModule_restrict_of_isIso_fromTildeΓ` derives section-localization *from* `[IsIso
fromTildeΓ]` — backwards for breaking circularity. The honest dependency is:
**G1-core (section-localization from QC data) → gap1 (IsIso fromTildeΓ) → general-scheme
`isLocalizedModule_basicOpen` (via U ≅ Spec Γ(X,U))**. Re-point the `\uses` edges accordingly;
the existing `isLocalizedModule_restrict_of_isIso_fromTildeΓ` becomes a downstream convenience,
not a prerequisite of gap1.
